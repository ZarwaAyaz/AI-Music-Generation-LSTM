# Music Generation with AI 

## Overview
This project generates piano music using Artificial Intelligence and LSTM neural networks trained on the MAESTRO MIDI dataset.

## Features
- MIDI music preprocessing
- Note extraction using music21
- Deep learning model using LSTM
- AI-generated music creation
- MIDI output generation

## Technologies Used
- Python
- TensorFlow
- music21
- NumPy
- Google Colab

## Dataset
MAESTRO MIDI Dataset

## Model
LSTM (Long Short-Term Memory)

## Output
The model generates new piano music and saves it as a MIDI file.

## Files Included
- Music_Generation_AI.ipynb
- generated_music.mid
- requirements.txt

## Future Improvements
- GAN-based music generation
- Transformer models
- Better melody control
- Multi-instrument generation
